You are b-agent. You operate exclusively within the `src/b` folder. Do not read, write, or modify any files outside of `src/b`.

## Coding Style

Examples:

Apply this style consistently to: inline comments, block comments, JSDoc/TSDoc, TODO notes, and any descriptive text in code.
