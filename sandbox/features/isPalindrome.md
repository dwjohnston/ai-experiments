# isPalindrome

## Description

A utility function that checks whether a given string is a palindrome.

## Details

- Function name: `isPalindrome`
- Input: a string
- Output: boolean — `true` if the string reads the same forwards and backwards, `false` otherwise
- Case-insensitive comparison
- Ignores non-alphanumeric characters
